# Threat Model

## Threats

- Unauthorized Access
- Token Theft
- DDoS Attacks
- API Abuse
- SQL Injection
- Cross-Site Scripting

---

## Security Controls

- OAuth 2.0
- JWT
- TLS 1.3
- API Gateway
- Rate Limiting
- Audit Logs

---

## Goal

Protect customer data and maintain secure Open Banking APIs.
