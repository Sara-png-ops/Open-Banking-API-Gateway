# Sandbox Environment

## Overview

The Sandbox Environment provides a safe testing platform for developers to integrate and test Open Banking APIs without accessing real customer data.

---

## Objectives

- Test APIs safely
- Simulate banking operations
- Validate OAuth authentication
- Test consent workflows

---

## Features

- Test customer accounts
- Mock transactions
- Sample balances
- Payment simulation
- Error simulation
- Rate limit testing

---

## Authentication

Supported methods:

- OAuth 2.0
- OpenID Connect
- JWT Tokens
- API Keys

---

## Test Data

The sandbox includes:

- Customer Profiles
- Bank Accounts
- Transaction History
- Payment Requests
- Consent Records

---

## Benefits

- No real financial data
- Safe API testing
- Faster developer onboarding
- Easier debugging
