# Open Banking API Gateway & Developer Portal Design

## Overview

This repository contains the design and documentation for an Open Banking API Gateway and Developer Portal developed as part of the Zetheta Algorithms Project Work Experience.

The project focuses on secure API management, consent management, OAuth 2.0 and FAPI security, developer onboarding, sandbox testing, monitoring, and OpenAPI-based API specifications.

---

## Features

- API Gateway Architecture
- OAuth 2.0 & Financial-grade API (FAPI) Security
- Consent Management Workflow
- Developer Portal Design
- Sandbox Environment
- OpenAPI Specifications
- Rate Limiting & API Versioning
- Monitoring & Observability
- Threat Modeling
